What is this website for?

This website documents my various hardware and software projects.

I have four hardware projects consisting of:
My homebrew CPUs: some CPUs I've tried to build over the years. The IRWIN 8 and RSSB CPUs being ones I designed myself.

MDA monitor: my attempt at finding an easy way to display text for the homebrew CPUs. ISA cards are far easier to talk to than creating my own GPU for composite, VGA etc.

Arduino IDE: a project about comunicating with an IDE superdisk floppy drive. Again this is an easier interface than directly controlling a regular floppy drive.

Random Tapes: these are just recordings from some tapes I've found. They contain things like music and some church sermon(I think?)

The software projects just link to my github and includes:
Monitor: functions for reading and writing to storage devices directly, by accessing it in sectors.

NTSB bot: a reddit bot for uploading the latest NTSB news to a subreddit.

FMAPI: converts data to sound that can be recorded to tape. It can then be played back and converted back into data. Essentially it makes a cassette tape a USB.

Youtube_DL_GUI: a GUI for a program called youtube-dl
